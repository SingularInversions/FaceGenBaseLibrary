Install

Path in which you have placed this sdk: ~sdk

FaceGen executables load data from ~sdk/data/ This directory must be present in the directory containing the executable, or any of its parent directories. It must contain the file _facegen_data_dir.flag

Windows

> PATH=%PATH%;~sdk\bin\win\x64\vs19\release\;~sdk\bin\win\x64\

Linux

$ PATH=$PATH:~sdk/bin/linux/x64/gcc/release/:~sdk/bin/linux/x64/

MacOS

$ PATH=$PATH:~sdk/bin/macos/x64/clang/release/:~sdk/bin/macos/x64/
